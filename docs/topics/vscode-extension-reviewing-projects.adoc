= Reviewing issues with the {ProductName} extension


You can review issues identified by the {ProductName} extension in *Issue Explorer*.

.Procedure

. In *Issue Explorer*, click on the project's folder and continue clicking until the issues are exposed.
+
Clicking on an issue reveals detailed information about a selected issue, including the hint and any additional resources.
+
To the left of each hint is an icon showing the severity of the issue and an indication if it possible to apply a "quick fix" to the problem.
+
The following icons indicate the severity and state of an issue:
+
.{ProductName} extension issue icons
[cols="20%a,80%",options="header",]
|====
|Icon |Description
|image::vs_mandatory.png[Mandatory] |The issue must be fixed for a successful migration.
|image::vs_optional.png[Optional or Quick Fix] |The issue is either optional to fix for migration _or_ it can be resolved by a "quick fix."

To determine which meaning is correct, right-click on the issue and look at the options in the list that opens. If *Apply All QuickFixes* appears, then a quick fix is available. Otherwise, fixing it is optional.
|image::vs_potential.png[Warning] |The issue might be an issue during migration.
|image::vs_resolved.png[Resolved] |The issue was resolved.
|====
+
. Click on a hint to reveal the lines of code that need to be changed.
. To see additional details about the issue, right-click on the issue and select *View details*.
